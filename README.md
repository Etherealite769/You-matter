# You Always Matter 💚

A beautiful, animated Mental Health Awareness presentation built with Node.js, Express, HTML, and CSS.

## Features

✨ **Glowing Animated Text** - Each letter pops in with a spinning, scaling effect
🌈 **Gradient Background** - Smooth color transitions for a calming effect
💫 **Pulse Circles** - Elegant pulsing elements in the background
📦 **Glassmorphism Design** - Modern frosted glass message box
💚 **Responsive Design** - Works perfectly on mobile and desktop

## Mental Health Awareness Month

This project celebrates Mental Health Awareness Month with an inspiring message: **You Always Matter**. Your mental health is important. Remember to reach out, take care of yourself, and support others.

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/you-matter.git
cd you-matter
```

2. Install dependencies:
```bash
npm install
```

3. Start the server:
```bash
npm start
```

4. Open your browser and navigate to:
```
http://localhost:5000
```
